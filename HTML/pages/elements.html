Block-Level Elements:

Block-level elements typically start on a new line and take up the full width available to them. They are used to structure the layout of a webpage.

  1. <div> : Defines a division or section of a document.
  2. <p> : Represents a paragraph of text.
  3. <h1> to <h6> : Headings of decreasing importance, where <h1> is the most important and <h6> is the least important.
  4. <header> : Represents introductory content at the beginning of a section or a webpage.
  5. <footer> : Represents footer of a section or a webpage.
  6. <nav> : Defines navigation bar or menu.
  7. <main> : Represents the main content of the document.
  8. <section> : Defines a section of content within a document.
  9. <article> : Represents a self-contained piece of content that could be distributed and reused independently.
  10. <aside> : Represents content that is tangentially related to the content around it, such as sidebars or pull quotes.
  11. <blockquote> : Represents a block of quoted text.
  12. <ul> : Defines an unordered list.
  13. <ol> : Defines an ordered list.
  14. <li> : Defines a list item within an ordered or unordered list.
  15. <table> : Defines a table.
  16. <form> : Defines a form for user input.
  17. <hr> : Represents a thematic break between paragraphs.
  18. <address> : Represents contact information for the author or owner of a document.

Inline Elements:

Inline elements do not start on a new line and only take up as much width as necessary. They are used to style and format text within a block-level element.

  1. <span> : Defines a small piece of text within a larger block of text.
  2. <a> : Creates a hyperlink to other web pages, files, locations within the same page, email addresses, or any other URL.
  3. <strong> : Indicates strong importance, typically rendered as bold.
  4. <em> : Indicates emphasis, typically rendered as italic.
  5. <img> : Embeds an image into the document.
  6. <br> : Inserts a single line break.
  7. <input> : Defines an input control, such as a text field or checkbox, within a form.
  8. <button> : Defines a clickable button.
  9. <label> : Defines a label for an <input> element.
  10. <textarea> : Defines a multiline text input control.
  11. <select> : Defines a dropdown list.
  12. <option> : Defines an option within a <select> element.
  13. <abbr> : Defines an abbreviation or acronym.
  14. <cite> : Represents the title of a work being cited.
  15. <code> : Defines a fragment of computer code.
  16. <mark> : Represents highlighted text.
  17. <sup> : Defines superscriptsup text.
  18. <sub> : Defines subscriptsub text.
