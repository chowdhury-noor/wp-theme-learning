Text Formatting Tags:

  1. <b> : Bold text.
  2. <strong> : Represents strong importance, typically displayed in bold.
  3. <i> : Italic text.
  4. <em> : Represents emphasized text, typically displayed in italics.
  5. <u> : Underlined text.
  6. <sup> : Superscript text (raised above the baseline).
  7. <sub> : Subscript text (lowered below the baseline ).
  8. <del> : Represents deleted or struck-through text.
  9. <ins> : Represents inserted text (typically underlined).
  10. <mark> : Highlights text with a background color.
  11. <abbr> : Defines an abbreviation or acronym with optional full form.
  12. <blockquote> : Defines a block quotation.
  13. <q> : Defines an inline quotation.
  14. <cite> : Defines the title of a work cited.
  15. <code> : Defines a piece of computer code.
