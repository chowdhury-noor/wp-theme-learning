HTML Form Tags

  1. <form> : Defines the start of the form.
  2. <label> : Defines a label for an input field.
  3. <input> : Defines an input field, such as text, email, or submit button.
  4. <textarea> : Defines a multiline text input field.
  5. <div> : Defines a division or section within the form for organizing elements.
  6. action="/submit-form" : Specifies the URL where the form data will be sent upon submission.
  7. method="post" : Specifies the HTTP method used to send the form data (POST method).
  8. for="name" , for="email" , for="message" : Associates labels with input fields using the for attribute.
  9. id="name" , id="email" , id="message" : Provides unique identifiers for input fields.
  10. name="name" , name="email" , name="message" : Provides names for input fields, which are used when submitting the form data.
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Form Example</title>
</head>
    
<body>
    
    <h2>Contact Form</h2>   
    
    <form action="/submit-form" method="post">
        <div>
            <label for="name"> Name:</label>
            <input type="text" id="name" name="name" required>
        </div>
        <div>
            <label for="email"> Email:</label>
            <input type="email" id="email" name="email" required>
        </div>
        <div>
            <label for="message"> Message:</label>
            <textarea id="message" name="message" row="4" required"></textarea>
        </div>
        <div>
            <input type="submit" value="submit">
        </div>
</body>

</html>

All Input Types

  1. Text

    : Allows users to enter a single-line text.

    <input type="text" name="username">

  2. Password

    : Similar to the text input but hides the entered characters for password entry.

    <input type="password" name="password">

  3. Email

    : Ensures the entered value is a valid email address.

    <input type="email" name="email">

  4. Number

    : Allows users to enter a number.

    <input type="number" name="quantity" min="1" max="10">

  5. Date

    : Provides a date input field.

    <input type="date" name="birthdate">

  6. Time

    : Allows users to enter a time.

    <input type="time" name="appointment_time">

  7. Checkbox

    : Allows users to select one or more options from a list.

    <input type="checkbox" name="interest" value="music"> Music
<input type="checkbox" name="interest" value="sports"> Sports

  8. Radio

    : Allows users to select only one option from a list.

    <input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

  9. File

    : Allows users to select and upload files.

    <input type="file" name="file_upload">

  10. Submit

    : Submits the form data to the server.

    <input type="submit" value="Submit">

  11. Reset

    : Resets the form fields to their default values.

    <input type="reset" value="Reset">

  12. Button

    : A clickable button with no default behavior.

    <button type="button">Click Me</button>

  13. Hidden

    : Allows a value to be passed to the server without displaying it to the user.

    <input type="hidden" name="token" value="abc123">

  14. Color

    : Allows users to select a color from a color picker.

    <input type="color" name="background_color">

  15. Range

    : Allows users to select a value from a range of values.

    <input type="range" name="volume" min="0" max="100">
