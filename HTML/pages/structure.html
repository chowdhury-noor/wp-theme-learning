Basic Web Page's HTML Structure

A basic HTML boilerplate provides a standard structure for an HTML document. It includes essential elements such as the document type declaration, HTML root element, head section, and body section. Here's an explanation of each part of a basic HTML boilerplate:

  1. Document Type Declaration (

    <!DOCTYPE html>

    ):

    • Defines the document type and version of HTML being used. In modern web development, <!DOCTYPE html> is used to specify HTML5.

  2. HTML Root Element (

    <html>

    ):

    • Represents the root of an HTML document.
    • Contains all other HTML elements.
    • The lang attribute specifies the language of the document (e.g., en for English, es for Spanish).

  3. Head Section (

    <head>

    ):

    • Contains meta-information about the document, such as its title, character encoding, viewport settings, and links to external resources.
    • Elements within the <head> section are not displayed on the web page itself.
    • Common elements within the <head> section include <meta> , <title> , <link> , and <style> .

  4. Body Section (

    <body>

    ):

    • Contains the visible content of the web page, such as text, images, links, forms, and other elements.
    • Elements within the <body> section are rendered and displayed in the web browser.

Here's a basic HTML boilerplate template that incorporates these elements:

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>

<body>
    <!-- Content goes here -->
</body>

</html>
