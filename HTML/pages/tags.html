List of HTML Tags

  1. <html> : Defines the root of an HTML document.
  2. <head> : Contains meta-information about the HTML document, such as its title, scripts, and stylesheets.
  3. <title> : Sets the title of the HTML document, which appears in the browser's title bar or tab.
  4. <base> : Specifies the base URL/target for all relative URLs in a document.
  5. <link> : Defines the relationship between the current document and an external resource, typically a stylesheet.
  6. <meta> : Provides metadata about the HTML document, such as character set, viewport settings, and keywords.
  7. <style> : Defines internal CSS styles within the HTML document.
  8. <script> : Embeds client-side scripts, such as JavaScript, within the HTML document.
  9. <body> : Contains the visible content of the HTML document.
  10. <header> : Represents introductory content, typically at the beginning of a section or a page.
  11. <footer> : Represents footer content, typically at the end of a section or a page.
  12. <main> : Represents the main content of the document, excluding header, footer and sidebar content.
  13. <nav> : Defines navigation links within the document.
  14. <article> : Represents self-contained content that can be independently distributed and reused.
  15. <section> : Represents a thematic grouping of content, typically with a heading.
  16. <aside> : Defines content that is tangentially related to the content around it, such as sidebars or pull quotes.
  17. <h1> to <h6> : Headings of decreasing importance, where <h1> is the most important and <h6> is the least important.
  18. <p> : Defines a paragraph of text.
  19. <a> : Creates a hyperlink to another web page or resource. Click here for more details on Anchor Tag
  20. <img> : Embeds an image within the document.
  21. <video> : Embeds video content within the document.
  22. <audio> : Embeds audio content within the document.
  23. <ul> : Defines an unordered (bulleted) list. Click here for more details on List Tags
  24. <ol> : Defines an ordered (numbered) list.
  25. <li> : Defines a list item within <ul> or <ol>.
  26. <dl> : Defines a description list, typically consisting of a series of term-description pairs.
  27. <dt> : Defines a term (name) in a description list.
  28. <dd> : Defines a description (value) in a description list.
  29. <table> : Defines a table for tabular data. Click here for more details on Table
  30. <tr> : Defines a row within a table.
  31. <th> : Defines a header cell within a table.
  32. <td> : Defines a data cell within a table.
  33. <form> : Defines an interactive form for user input.
  34. <input> : Defines an input control, such as a text field, checkbox, or radio button.
  35. <textarea> : Defines a multiline text input control.
  36. <button> : Defines a clickable button.
  37. <label> : Defines a label for an <input> element.
  38. <select> : Defines a dropdown list.
  39. <option> : Defines an option within a <select> element.
  40. <optgroup> : Groups related <option> elements within a <select> element.
  41. <iframe> : Embeds a nested browsing context, such as another HTML page, within the current document.
  42. <canvas> : Defines an area for drawing graphics using JavaScript.
  43. <details> : Defines additional details or information that can be toggled open or closed.
  44. <summary> : Defines a summary or caption for the <details> element.
  45. <blockquote> : Defines a block quotation, typically indented and styled differently from surrounding text.
  46. <cite> : Defines the title of a work cited within a <blockquote> or <q> element.
  47. <q> : Defines an inline quotation.
  48. <abbr> : Defines an abbreviation or acronym.
  49. <address> : Defines contact information for the author or owner of a document.
  50. <time> : Defines a specific time or date.
