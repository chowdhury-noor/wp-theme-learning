HTML List Tags

  1. Unordered List ( <ul> ):

    An unordered list is a list of items marked with bullets.

    • <ul> : Defines the start of an unordered list.
    • <li> : Defines a list item within the unordered list.

  2. Ordered List ( <ol> ):

    An ordered list is a list of items marked with numbers or letters.

    • <ol> : Defines the start of an ordered list.
    • <li> : Defines a list item within the ordered list.

  3. Description List ( <dl> ):

    A description list consists of term-definition pairs.

    • <dl> : Defines the start of a description list.
    • <dt> : Defines a term (name) in the description list.
    • <dd> : Defines a description (value) associated with the term.
