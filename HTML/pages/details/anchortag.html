HTML Link Tag

  1. Anchor <a> Tag:

    The <a> tag is used to create hyperlinks. It has two essential attributes:

    • href : Specifies the URL of the page the link goes to. It can be an absolute URL (e.g., https://example.com ) or a relative URL (e.g., about.html , #section ).
    • target : Specifies where to open the linked document. Common values include _self (open in the same window or tab), _blank (open in a new window or tab)

      • Example

      :

      Here's an example of an HTML link tag:

      <a href="<https://www.example.com>" target="_blank">Visit Example</a>

  2. Linking to Sections within a Page:

    You can use links to navigate to specific sections within the same page by using the id attribute on elements you want to link to, and then using the # symbol followed by the id value as the href attribute in the link.

    <!-- Target Section -->
<h2 id="section">Section Title</h2>

<!-- Link to the Target Section -->
<a href="#section">Go to Section</a>
