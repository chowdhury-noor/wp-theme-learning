HTML Table Tags

  1. <table> : Defines the start of the table.
  2. <thead> : Defines the header section of the table.
  3. <tbody> : Defines the body section of the table.
  4. <tr> : Defines a row in the table.
  5. <th> : Defines a header cell in the table (typically used in the header row).
  6. <td> : Defines a data cell in the table (typically used in the body rows).
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Table Example</title>
</head>

<body>

    <h2>Employee Details</h2>   
    
    <table>
        <thead>
            <tr>
                <th>ID</th>
                <th>Name</th>
                <th>Position</th>
                <th>Salary</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <th>1</th>
                <th>John Doe</th>
                <th>Software Engineer</th>
                <th>$60,000</th>
            </tr>            
            <tr>
                <th>2</th>
                <th>Jane Smith</th>
                <th>Project Manager</th>
                <th>$75,000</th>
            </tr>            
            <tr>
                <th>3</th>
                <th>Michael Johnson</th>
                <th>UX Designer</th>
                <th>$65,000</th>
            </tr>            
        </tbody>

    </table>

</body>

</html>
